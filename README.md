# EPV Horizons Savants

Plateforme d'excellence éducative à Abidjan — École Maternelle et Primaire d'Excellence.

## Lancer en local

**Prérequis :** Node.js

1. Installer les dépendances :
   `npm install`
2. Copier `.env.example` vers `.env.local` et renseigner les clés API (Resend, Twilio…)
3. Démarrer le serveur :
   `npm run dev`

Le serveur affiche au démarrage l'adresse locale **et** l'adresse réseau WiFi pour y accéder depuis un autre appareil.
